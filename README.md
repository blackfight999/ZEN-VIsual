# Serenity - Mindful Happiness Platform

A zen-inspired web experience designed to help you find stillness in a chaotic world. Built with React, Three.js, and Tailwind CSS.

## About

Serenity is a personal project exploring the intersection of mindfulness, modern web technologies, and visual design. The site features calming earth tone pastels, interactive breathing exercises, and wisdom from philosophers like Alan Watts.

## Features

- **Breathing Pacer** - Interactive breathing exercises to calm your nervous system
- **Gratitude Garden** - Daily gratitude practice tracker
- **3D Zen Scenes** - Calming visual environments built with Three.js
- **Alan Watts Wisdom** - Curated quotes on presence, acceptance, and flow
- **Responsive Design** - Beautiful earth tone palette that works on all devices

## Tech Stack

- React 19
- TypeScript
- Vite
- Three.js / React Three Fiber
- Framer Motion
- Tailwind CSS
- Lucide Icons

## Getting Started

Prerequisites: Node.js 20+

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The site will be available at `http://localhost:3000`

## Deployment

This project is configured for GitHub Pages deployment. Push to `main` branch to trigger automatic deployment via GitHub Actions.

Live site: https://blackfight999.github.io/ZEN-VIsual/
